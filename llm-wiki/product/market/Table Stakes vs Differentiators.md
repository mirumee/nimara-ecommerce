## Content

| Capability Area | Commoditized (Table Stakes) | Genuine Differentiator (Nimara's Opportunity) |
| :-- | :-- | :-- |
| **Core Commerce Logic** | Carts, checkout, inventory counts, basic order processing | Decoupled, swappable backend engines (e.g. swap Saleor for Medusa) without rewriting storefront code |
| **Localization** | Multi-currency display, language selectors, translation tables | Native automated cross-border compliance (EU Omnibus pricing, localized tax) pre-integrated |
| **Search & Discovery** | Basic keyword search and simple DB filtering | Deep real-time Meilisearch/Algolia configs with zero-config admin dashboards |
| **Deployment DX** | Docker containers, standard env var templates | Single-command "Zero-to-Deploy" — serverless infra, DB seeding, hosting in under 10 minutes |
| **Marketplace Support** | Simple vendor attributes on product listings | Open-source Stripe Connect split-payment orchestration with automated commissions and per-vendor payouts |
| **AI Integration** | Chatbots that recommend products or generate search keywords | Exposed Model Context Protocol (MCP) endpoints and Universal Commerce Protocol (UCP) compliance for AI agents |

### How to read this
- The left column is the credibility floor — missing any of it loses adopters before they evaluate the differentiators.
- The right column maps directly to the [[Initiative Prioritization]] list. Each differentiator is the rationale behind a prioritized initiative.
