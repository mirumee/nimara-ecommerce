# Documentation Overview

Welcome to the Nimara documentation! This guide will help you set up and deploy your e-commerce storefront using Nimara, a modern headless e-commerce platform built on top of Saleor.

## Getting Started

To get started with Nimara, explore the **Quickstart** section which contains step-by-step guides for:

- **Running locally** - Set up Nimara on your local development environment
- **Storefront** - Deploy your storefront to production
- **Environment variables** - Configure your application settings
- **Stripe integration** - Set up payment processing
- **Saleor CMS** - Manage your content with Saleor's CMS features
- **Saleor channels** - Add and configure new sales channels
- **Terraform** - Infrastructure as code deployment

Choose a guide from the menu to begin!
