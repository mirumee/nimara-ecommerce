# Deploying Nimara to Vercel using Terraform

Nimara can be deployed to Vercel using Terraform, which allows you to manage your infrastructure as code. This guide will walk you through the steps to deploy Nimara to Vercel using Terraform.

### ⚙️ Prerequisites

Before proceeding, ensure you have:

- Installed [Terraform](https://developer.hashicorp.com/terraform/install)
- Access to your **Vercel account**
- Access to your **Saleor Dashboard**
